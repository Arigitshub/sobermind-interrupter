1. The Dopamine Loop

Dopamine is a neurotransmitter responsible for reinforcement learning, reward prediction, and motivation. Modern application design exploits this mechanism by constructing short, unpredictable habit loops:

"Trigger ➔ Craving ➔ Automatic Response ➔ Variable Reward"

By offering infinite scroll layouts and intermittent notifications, applications hijack this sequence, leading to involuntary autopilot usage. SoberMind inserts a structural delay between the Trigger and the Response, neutralizing the chemical hook.

2. The 90-Second Rule

According to neuroanatomists, when an impulse is triggered, the physiological cascade that creates the bodily urge (such as increased heart rate, shallow breathing, and mental tension) has a finite lifespan of approximately 90 seconds.

If you refrain from reinforcing the urge with automatic thoughts or rationalizations during this window, the hormone surge peaks, returns to baseline, and dissolves. The timer coordinates your attention to help you pass this peak.

3. Grounding Practices

SoberMind guides your focus during the 90 seconds through specific, clinically backed methods:

  • 4-7-8 Breathing: Stimulates the vagus nerve, initiating parasympathetic calm.
  • 5-4-3-2-1 Senses check: Disrupts cyclic looping thoughts by forcing tactile attention onto physical environment signals.
  • Reframing questions: Activates prefrontal executive function to counter irrational dopamine justifications.

4. Shell & Terminal Alias Integration

Developers spend significant time inside command terminals. You can configure shell aliases to route command lines through the SoberMind pause timer. For example, if you run the shell alias `check` instead of opening websites directly, it forces a mindfulness delay:

# Add this to your ~/.zshrc or ~/.bashrc file
function check() {
  echo "Mindfulness delay active..."
  open "https://sobermind.surge.sh"
}

5. Terminal Command: `sobermind`

You can run the SoberMind interrupter directly from your command terminal as a global command. This allows you to pause automatic actions inside your terminal workflow.

Option A: Install via Pip (Official PyPI Release)

SoberMind is published globally on the Python Package Index. Install it directly on your system with one command:

pip install sobermind

Once installed, execute the command from any directory or terminal session:

sobermind "browsing social media" --duration 90

Option B: Windows PATH / Batch script

For Windows command prompts, you can add the project directory to your system environment PATH variable. The directory contains a batch script wrapper sobermind.bat which executes Python directly:

# Execute directly in cmd or PowerShell
sobermind "boredom snack urge"
